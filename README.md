# nodejs-rabbitmq

`rabbitmq` 是一种在进程间，应用间和服务器之间交换数据的方式。

1.  通过`rabbitmq`进行通信的应用程序可以是不同编程语言开发的

2.  `rabbitmq` 通过`pub-sub`模式对生产者和消费者进行了解耦
