# `Mac OSX`系统使用`brew`安装`rabbitmq`，日志、插件，配置文件，可执行文件等路径

## 安装

OSX:

```bash
brew install rabbitmq
```

##  路径

| Name                 | Location                    |
| -------------------- | --------------------------- |
| RABBITMQ_CONFIG_FILE | /usr/local/etc/rabbitmq     |
| RABBITMQ_LOG_BASE    | /usr/local/var/log/rabbitmq |
|  可执行文件          | usr/local/sbin              |
